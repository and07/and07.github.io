Convert CSV To SQL

Use this tool to convert CSV to SQL statements.
INSERT, UPDATE, and DELETE statements can be created. CREATE TABLE and DROP TABLE are optional. The key used in UPDATE and DELETE is specified by setting the key column. You can specify which fields to include and specify the name of the field. A field value may be trimmed, made uppercase, or lowercase. You can add custom text around the field value by using the template feature. For example, if you are using Oracle and want to convert a field in YYYYMMDD format to DATE, use TO_DATE({f},'YYYYMMDD'). The field value will be substituted for {f}.
Current version supports some database specific output and support ORACLE, MYSQL, SQL SERVER and most ANSI databases. There are slight variations in the way different databases handle NULLs, empty strings, DATES, etc. .
